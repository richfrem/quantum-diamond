# Contributing (Historical Reference)

This repository is now in maintenance mode and has been superseded by the [Quantum Diamond Forge](https://github.com/richfrem/quantum-diamond-forge).

## For New Contributions

All new development, features, and improvements should be directed to the Quantum Diamond Forge repository:

- **Repository**: https://github.com/richfrem/quantum-diamond-forge
- **Issues**: https://github.com/richfrem/quantum-diamond-forge/issues
- **Discussions**: https://github.com/richfrem/quantum-diamond-forge/discussions

## For Critical Bug Fixes

If you've discovered a critical security issue or bug in this deprecated framework:

1. First check if the issue exists in [Quantum Diamond Forge](https://github.com/richfrem/quantum-diamond-forge)
2. If it's specific to this deprecated version, open an issue describing the problem
3. Critical security issues will be addressed; feature requests will not

## Migration Support

If you need help migrating from this framework to the Forge:

- See [MIGRATION.md](./MIGRATION.md) for detailed migration steps
- Open a discussion in the [Forge repository](https://github.com/richfrem/quantum-diamond-forge/discussions)
